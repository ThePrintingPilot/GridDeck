Instructions

How to install GridDeck

  1. Use Chrome or Edge on desktop.
  2. Connect the GridDeck screen using a USB data cable.
  3. If the device is not detected, hold BOOT while pressing RESET, then try again.
  4. Click Connect Device, choose the ESP32-S3 port in the browser prompt, then click Install / Flash.
  5. Do not disconnect the cable until the installation finishes.
